Show how you've been goofing off on the Mac Desktop.

To run:
python whatchadoin.py /tmp/whatchadoin.png

Currently, if you want to change its behavior (categories, colors,
etc.) you need to edit the code.

Then add a <a
href="http://projects.tynsoe.org/en/geektool/>GeekTool</a> pointing to
that image to your desktop.

Prerequisistes: AppKit, PIL.
